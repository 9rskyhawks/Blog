Monitoring Performance with the PageSpeed Insights API

Optimize Your Website for Speed and User Experience

What is the PageSpeed Insights API?

The PageSpeed Insights (PSI) API is a tool provided by Google that analyzes the performance of your website. It provides a score and actionable insights to optimize your website for speed and user experience.

Why Use the PSI API?

  • Identify performance bottlenecks.
  • Receive detailed suggestions for optimization.
  • Track performance changes over time.
  • Automate performance monitoring for your projects.

Step 1: Get API Access

To use the PSI API, you’ll need to set up an API key:

  1. Go to the Google Cloud Console.
  2. Create a new project or select an existing one.
  3. Enable the PageSpeed Insights API.
  4. Generate an API key under the “APIs & Services” section.

Step 2: Make an API Request

Here’s an example of an API request using curl:

curl "https://www.googleapis.com/pagespeedonline/v5/runPagespeed?url=https://example.com&key=YOUR_API_KEY"

The response will include:

  • Performance Score: A numeric score (0–100).
  • Core Web Vitals: Metrics like LCP, FID, and CLS.
  • Opportunities: Suggestions to improve performance.

Step 3: Parse the API Response

The API returns a JSON response. Here’s an example of parsing the data in JavaScript:

const fetch = require('node-fetch');

const apiKey = 'YOUR_API_KEY';
const url = 'https://example.com';

fetch(`https://www.googleapis.com/pagespeedonline/v5/runPagespeed?url=${url}&key=${apiKey}`)
  .then(response => response.json())
  .then(data => {
    console.log('Performance Score:', data.lighthouseResult.categories.performance.score * 100);
    console.log('First Contentful Paint:', data.lighthouseResult.audits['first-contentful-paint'].displayValue);
  })
  .catch(error => console.error('Error:', error));

Step 4: Use Insights to Improve Performance

Common recommendations from PSI include:

  • Minify CSS, JavaScript, and HTML files.
  • Optimize and compress images.
  • Enable browser caching and use a CDN.
  • Implement lazy loading for images and videos.
